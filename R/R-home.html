Learning R

R is a programming language for statistical computing and graphics supported by the R Core Team and the R Foundation for Statistical Computing. The R programming language gets used by many quantitative analysts as a programming tool since it's useful for data importing and cleaning.

This module will cover visualizations such as bar plots, histograms, pie charts, line plots, and scatterplots. I will be focusing on the ggplot library since I believe it is the best for customization and appearance.

This module will cover what is R Shiny and how to make a basic app using R.
